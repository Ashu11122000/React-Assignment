# React Practice Assignments 🚀

A collection of beginner-to-intermediate ReactJS projects designed to practice the most important React concepts including **Components, JSX, Props, State, Events, Forms, Hooks, Conditional Rendering, API Integration, Routing, and Performance Optimization**.

---

# 🛠 Tech Stack

- ReactJS
- JavaScript (ES6+)
- Vite
- Tailwind CSS
- HTML5
- CSS3
- OpenWeatherMap API
- React Router DOM

# Getting Started

## Clone Repository

```bash
git clone <repository-url>
```

---

## Navigate into Project

```bash
cd react-practice-assignments
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

Open your browser at

```
http://localhost:5173
```
